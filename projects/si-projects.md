## 1. 해외 뷰티 쇼핑몰 (theBeautyzon)

> 2021.12 ~ 2022.05 · PHP

### 개요
eBay를 벤치마킹한 글로벌 쇼핑몰. 회원/셀러/관리자 3단계 권한,
경매·이벤트·상품·결제·기프트카드로 구성. 상품·결제·배송 영역을 담당했다.

### 핵심
- 결제(Stripe, PayPal), 배송(Shippo), 미국 세금(Avalara) API를 **전부 직접 연동**
- 특히 Avalara·Shippo는 당시 국내 연동 사례가 없어,
  **해외 공식 문서만 보고 직접 테스트하며 연동을 완료**
- 캐싱 적용으로 페이지 로딩 속도 개선 (속도 측정 기반으로 개선 확인)

## 2. 공임나라

> 2021.04 ~ 2021.08 · PHP

### 개요
ASP/MSSQL 기반 레거시를 PHP/MariaDB로 이관하는 프로젝트.
언어 전환과 이기종 DB 이관이 동시에 필요했다.

### 핵심
- 회원 **80만 명 이상** 규모를 무중단으로 이관
- **PHP 이관 스크립트를 직접 작성**하여 데이터를 옮기며 정합성을 검증
- 기존 DB를 운영 상태로 유지한 채 약 1주간 이관 작업 후 DB 교체로 무중단 전환
