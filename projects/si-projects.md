# SI Projects (WebBiz)

## 📌 개요
다양한 도메인의 SI 프로젝트를 수행하며  
요구사항 분석부터 설계, 개발, 배포, 운영까지 전 과정을 경험

---

## 1. 통계청 대표 홈페이지 개선

### 문제
- 검색 성능 저하 및 사용자 경험 문제

### 해결
- SQL 튜닝 및 인덱스 최적화
- 검색 API 연동 개선

### 결과
- 검색 성능 30% 개선

---

## 2. 해외 뷰티 쇼핑몰 (theBeautyzon)

### 문제
- 글로벌 결제 및 물류 연동 필요 및 신규 개발 필요
- 페이지 로딩 속도 저하

### 해결
- PayPal, Stripe, Shippo API 연동
- 캐싱 적용 및 구조 개선

### 결과
- 신규 서비스 런칭
- 페이지 로딩 속도 40% 개선
- 글로벌 서비스 안정성 확보

---

## 3. 공임나라

### 문제
- 기존 DB 구조 및 시스템 한계

### 해결
- MSSQL → MariaDB 마이그레이션
- 예약 및 결제 시스템 개선

### 결과
- 안정적인 예약/결제 시스템 구축

---

## 4. Celuvu

### 문제
- 초기 서비스 성능 및 구조 설계 필요

### 해결
- DB 구조 설계 및 캐싱 전략 적용
- PHP 사용한 서비스 개발

### 결과
- 이미지 로딩 속도 45% 개선
- 안정적인 서비스 런칭
