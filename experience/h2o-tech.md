# Core Backend Experience (에이치투오코리아)

숙박·레저 커머스 플랫폼에서 레거시 개선, 신규 개발, 인프라 구축을 두루 담당했다.
아래는 핵심 경험을 영역별로 정리한 것이다.

---

## 📌 레거시 시스템 고도화 (PHP / PHP→Java)

### 문제
PHP 기반 레거시 코드의 복잡한 구조와 반복 장애. 성수기에는 서버가 하루 2~3회 다운.

### 해결
- PHP cURL 소켓 정리, 좀비 프로세스 정리, DB 메모리 누수 추적·정리
- OMS의 PHP+Java 혼용 코드를 Java로 통합 마이그레이션

### 결과
- 성수기 무중단 운영 달성
- 동급 스펙 기준 서버 자원 약 40%, 비용 20~30% 절감
- 구조 일관성 및 유지보수성 확보

---

## 📌 신규 기능 개발 (Java, Spring Boot)

### 역할
- 다올비전 CMS 양방향 연동 (객실·요금 수신, 외부 채널 결제건 송신) — Redis 활용
- 카카오 예약하기 상품·예약결제 양방향 API 연동 (최대 초당 수백 건 동기화)
- OMS 수수료 정산 시스템 개발: 하드코딩되어 있던 60여 종 수수료 로직을
  관리자가 수식으로 직접 관리하면 입금가·판매가·정산가가 자동 산출되도록 개선

### 결과
- 외부 채널 연동으로 신규 판매 채널 확보
- 비즈니스 로직의 설정 데이터화로 운영 효율·유지보수성 향상

---

## 📌 이미지 업로드 서버 (Kotlin Coroutine)

### 문제
여러 시설에서 동시에 대량 이미지가 업로드되는 환경. 블로킹 처리 시 스레드 고갈 위험.

### 해결
- S3 + CloudFront 기반 업로드 서버 구축
- Kotlin Coroutine 논블로킹 비동기 처리로 동시 다발 I/O 부하 대응

### 결과
- 이미지 로딩 속도 약 2배 개선(실측), 인프라 비용 절감
- 프로모션 시 이미지 누락 문제 완전 해소
